🔧 Firestore 規則助手

避開手機貼上「隱形字元」問題,一鍵複製乾淨規則

1 複製規則

按下面按鈕,規則會複製到剪貼簿(絕對乾淨、無隱形字元)

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}

2 開啟 Firebase Console

會在新分頁打開 yuanziifamily 的規則頁

3 貼上規則(在 Firebase 頁面裡)

a. 點編輯區任何位置
b. 長按 → 全選
c. 按 刪除(清成空白)
d. 長按空白處 → 貼上(這次是乾淨的了)
e. 按右上 發布

4 測試
